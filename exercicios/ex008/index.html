Principais formatações

Negrito / Destaque

Nesta frase temos um termo em negrito usando a tag < b >. A tag < b > é uma tag não semântica.

Nesta frase temos um termo em negrito usando a tag < strong >. A tag < strong > é uma tag semântica.

Itálico / Ênfase

Nesta frase temos um termo em itálico usando a tag < i >. A tag < i > não é semântica.

Nesta frase temos um termo em Ênfase usando a tag < em >. A tag < em > é uma tag semântica.

Texto Marcado

Podemos criar também um texto marcado usando a tag < mark >

Texto grande e pequeno

Estamo criando um texto grande e um texto pequeno este parágrafo.

Texto deletado

Podemos marcar um texto como excluído para indicar que ele deve ser lido, mas não considerado usando a tag < del >

Texto inserido

Podemos marcar um texto como inserido para dar uma ênfase e indicar que ele foi adicionado depois usando a tag < ins >

Texto Sobrescrito e Texto Subscrito

Calcule: x20 + 2.

Para fazer o "20" subir foi usada a tag < sup >.

Formula da água H20

Para fazer o "2" descer foi usada a tag < sub >.
